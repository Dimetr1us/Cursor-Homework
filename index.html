Перелік основних команд Git

  • git config --global user.name "Username" - налаштування імені користувача
    • .
  • git config --global user.email "Username@mail - налаштування електронної пошти користувача

  • git init - створення порожнього Git репозиторію, або реініціалізація існуючого

  • git status - показує статус файлів робочого дерева

  • git add - додає обрані файли до наступного комміта

  • git commit - записує зміни до репозиторію

  • git log - показує історію змін комміта

  • git checkout - змінює гілку, або повертає файли робочого дерева

  • git branch - список, створення або видалення гілок

  • git push - завантажує вмісту локального репозиторію до віддаленого репозиторію

  • git pull - завантажує та інтеграція з віддаленого репозиторія або іншої гілки з поточною гілкою

  • git merge - об'єднує дві абобільше гілок разом

  • git clone - клонує вміст репозиторія у нову директорію

.gitignore файл - цілеспрямованно вказує ті файли та папки, які слід скриваті від системи контроля версій Git

Advanced

  • git revert -не видаляє, а скасовує зміни, внесені в комміт та додає новий з отриманим вмістом

  • git reset -повертає головну гілку до попереднього, обраного стану

  • git rebase -Повторно використовує комміти для з'єднання на іншому кінці головної гілки

  • git cherry-pick -використовує зміни, впроваджені іншим існуючим коммітом для повторного використання

  • git stash -прихованно зберігає зміни до сховища незавершених змін
